# くりさがり ひきざん ドリル 🐿️

2桁の繰り下がりのある引き算を、iPad で楽しく練習できる Web アプリです。

## あそびかた

- **ひっさんモード** … 学習指導要領(小2)の筆算の手順どおりに解きます
  1. 一の位が引けない → 十の位の数字を**タッチして繰り下げ**(斜線が引かれ、1減った数が上に書かれます)
  2. 「10いくつ − ○」で一の位を計算して入力
  3. 繰り下げたあとの十の位を計算して入力
- **こたえだけモード** … 暗算で答えだけを入力する練習モード

数字の入力はすべて画面上の 0〜9 の大きなボタンで行うので、iPad でのタッチ操作にぴったりです。1回 10問で、最初から正解できた問題の数だけ ⭐ がもらえます。

## 公開ページ(GitHub Pages)

GitHub Pages を有効にすると、以下の URL からアクセスできます:

**https://flibrariant.github.io/sansu_kuri/**

### Pages の有効化手順(初回のみ)

1. GitHub のリポジトリページで **Settings → Pages** を開く
2. 「Build and deployment」の Source を **Deploy from a branch** にする
3. Branch に **main** / フォルダ **/(root)** を選んで Save
4. 1〜2分待つと上の URL でアクセスできるようになります

iPad の Safari で開いて「ホーム画面に追加」すると、アプリのように使えます。
